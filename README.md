# Hacknarok 2026

## Jak zainstalować?:
- Pobierz repozytorium
- Wejdź pod ten adres : `chrome://extensions/`
- Włącz tryb dewelopera
- Wybierz `Załaduj rozpakowane`
- Wybierz folder o nazwie `Extension` z repozytorium
- Przyznaj trwały dostęp do mikrofonu oraz kamery dla wtyczki
- Pobierz id wtyczki z menu pluginów i wejdź pod adres: `chrome-extension://<id wtyczki>/offscreen.html`
