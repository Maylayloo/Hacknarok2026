# Hacknarok 2026

## Jak zainstalować?:
- Pobierz repozytorium
- Wejdź pod ten adres : `chrome://extensions/`
- Włącz tryb dewelopera
- Wybierz `Załaduj rozpakowane`
- Wybierz folder o nazwie `Extension` z repozytorium
- Pobierz unikalny kod z menu pluginów i wejdź pod adres: `chrome-extension://<kod dla wtyczki>/offscreen.html`
- Wtyczka gotowa do użycia
